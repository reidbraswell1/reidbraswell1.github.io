HTML Editors

The 2 html editors that we will be using in the class are HTML Code Editor and Online HTML Editor . The first editor allows you to write your document in HTML in the left pane and see the result in the right pane. The second editor allows you to edit a document in with a word like word processor and see the HTML result in the right pane. You can also make modifications on the right pane and have them reflected in the left pane.

HTML

Definition

HTML (Hypertext Markup Language) forms the basic skeleton of a web-page.

History

Late 1991 the first public description of HTML was revealed known as "HTML TAGS" by Tim Burnes Lee. 1

Apperance

A basic minimal HTML skeleton would look like this:

<html lang="en">
  <head>
  <title>Your Page Title</title>
  </head>
  <body>
    <nav>
      <!-- Basic Navigation Links For the Document -->
    </nav>
    <main>
      <!-- Main Content For the Document -->
    </main>
    <footer>
      <!-- Footer For the Document (copyright etc...) -->
    </footer>
  </body>
</html>

CSS (Cascading Style Sheets)

Definition

Cascading Style Sheets is a style sheet language that describes how HTML elements are to be displayed in an HTML document. A cascading style sheet is normally placed in a seperate file with a css extension.

History

The initial release was December 17, 1996 and was developed by HÃ¥kon Wium Lie, Bert Bos and the World Wide Web Consortium (W3C).

Appearence

CSS to style a simple heading element with a color of red and text centered would look like this:

h3 {
  color: red;
  text-align: center;
}
CSS Selectors

CSS selectors enable us to select an element for styling. For the class the main ones we will be working with are the element selector, element.class selector and element element selector. The element selector selects all elements with that tag name. For example p selects all <p> elements for styling. The element.class selector selects all elements with that class name. For example p.end selects all <p class="end"> elements for styling. The element element selector selects all elements that have a parent as the first element. For example nav a selects all <a> elements that are inside of a <nav> element.

For a complete listing of selectors see CSS Selectors and CSS Selectors MDN

Definition

For example how the elements are laid out, navigation and interaction. Wireframes are usually void of style and color. A wireframe can be produced on paper, white board or with commercial software.

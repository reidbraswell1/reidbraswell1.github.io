HTML

HTML forms the basic skeleton of a web-page. A basic minimal HTML skeleton would look like this: 

<html>
    <head>
        <title>Your Page Title</title>
    </head>
    <body>
        <nav>
            <!-- Basic Navigation Links For the Document -->
        </nav>
        <main>
          <!-- Main Content For the Document -->
        </main>
        <footer>
          <!-- Footer For the Document (copyright etc...) -->
        </footer>
    </body>
</html>

HTML Editors

The 2 html editors that we will be using in the class are HTML Code Editor and Online HTML Editor . The first editor allows you to write your document in HTML in the left pane and see the result in the right pane. The second editor allows you to edit a document in with a word like word processor and see the HTML result in the right pane. You can also make modifications on the right pane and have them reflected in the left pane.

HTML

Definition

HTML (Hypertext Markup Language) forms the basic skeleton of a web-page.

History

Late 1991 the first public description of HTML was revealed known as "HTML TAGS" by Tim Burnes Lee. 1

Apperance

A basic minimal HTML skeleton would look like this:

<html>
  <head>
  <title>Your Page Title</title>
  </head>
  <body>
    <nav>
      <!-- Basic Navigation Links For the Document -->
    </nav>
    <main>
      <!-- Main Content For the Document -->
    </main>
    <footer>
      <!-- Footer For the Document (copyright etc...) -->
    </footer>
  </body>
</html>
